# Algo Problems Java

A Java-based project for learning and practicing algorithms and data structures.

## Tech Stack

- **Java 25**
- **Maven** for build management
- **JUnit 6** + **AssertJ** for testing

## Project Structure

```
src/main/java/com/samples/
├── algo/easy/          # Basic algorithm implementations
│   ├── NumberPalindrome
│   ├── StringPalindrome
│   ├── SetMismatch
│   └── ValidParentheses
├── datastruct/simple/  # Core data structure implementations
│   ├── Queue (Array & LinkedList implementations)
│   ├── Stack (Array & LinkedList implementations)
│   └── Node
└── common/             # Shared utilities
    └── Person
```

## Getting Started

**Build:**
```bash
mvn clean compile
```

**Run Tests:**
```bash
mvn test
```
